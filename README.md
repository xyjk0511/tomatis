# Tomatis 训练管理控制台（前端原型）

这是一个基于 Vite + React + Tailwind 的前端原型，用于模拟 Tomatis 训练场景下的设备监控、患者档案管理、训练方案管理以及素材库管理。项目以本地 mock 数据运行，便于 UI/流程演示与二次开发。

## 功能概览
- 设备监控中心：12 通道设备卡片、运行/暂停/恢复/停止、批量启动与暂停、音量控制与训练进度计时
- 登录与安全：演示用账号、首次登录改密流程、登录失败锁定机制
- 患者档案：新增/编辑/删除、搜索与分页、治疗记录筛选、导出 PDF/Excel 报告
- 方案管理：预置/自定义方案、复制/编辑/删除、阶段参数编辑、方案 JSON 导出
- 素材库：目录树管理、搜索与排序、批量操作、音频预览、录音与导入、导出到 Audacity
- 系统设置：Audacity 路径配置（演示用）

## 技术栈
- React 18 + Vite
- Tailwind CSS
- jsPDF / xlsx（报告导出）

## 快速开始
```bash
npm install
npm run dev
```

构建与预览：
```bash
npm run build
npm run preview
```

## 目录结构
- `App.jsx`：应用入口与主状态管理
- `views/`：各页面视图（监控中心、患者档案、方案管理、素材库、设置）
- `components/`：通用组件与弹窗
- `data/mockData.js`：mock 数据与默认方案配置
- `utils/`：工具函数与全局样式

## 说明
- 当前为前端原型，数据存储在内存中，刷新页面会重置。
- 录音、导出、文件移动等部分为交互演示逻辑，未对接真实文件系统或后端服务。
